# <a name="manage-app-groups-by-using-the-azure-portal-for-windows-virtual-desktop"></a>Administrar grupos de aplicaciones con Azure portal para escritorio virtual de Windows

El grupo de aplicaciones predeterminado creado para un nuevo grupo host de escritorio virtual de Windows también publica el escritorio completo. Además, el uso del portal de Azure le permite crear uno o más grupos de aplicaciones de RemoteApp para el grupo de hosts.

El proceso de implementación hará lo siguiente:

1. Cree el grupo de aplicaciones RemoteApp.
2. Agregue las aplicaciones seleccionadas al grupo de aplicaciones.
3. Publique usuarios individuales o grupos de usuarios en el grupo de aplicaciones.
4. Registre el grupo de aplicaciones, si decide hacerlo.
5. Cree un vínculo a una plantilla de Azure Resource Manager de acuerdo con la configuración, que puede descargar y guardar.

Para crear un grupo de RemoteApp para el escritorio virtual de Windows, siga las instrucciones que se indican en [Manage App Groups with the Azure portal](https://go.microsoft.com/fwlink/?linkid=2129550).
