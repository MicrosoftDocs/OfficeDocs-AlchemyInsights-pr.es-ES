# <a name="set-up-microsoft-information-protection"></a>Configurar la protección de la información de Microsoft

Obtenga información general sobre las funcionalidades que puede usar con su estrategia de protección de la información para asegurarse de que la información confidencial esté protegida. Aplique un planteamiento de ciclo de vida de cuatro fases con el que detectar, clasificar, proteger y supervisar la información confidencial. La [Guía de configuración de Microsoft Information Protection](https://go.microsoft.com/fwlink/?linkid=2146619) contiene instrucciones para completar cada una de esas fases.
